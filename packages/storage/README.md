# `@glocurrency/storage`

Tiny utilities to make use of local storage

---

## Install

```bash
$ pnpm add @glocurrency/storage
```

## Usage

```js
import { isLocalStorageAvailable } from '@glocurrency/storage'

const available = isLocalStorageAvailable()
```
