# `@glocurrency/uuid`

UUID Helpers

---

## Install

```bash
$ pnpm add @glocurrency/uuid
```

## Usage

```js
import { first8 } from '@glocurrency/uuid'

first8('c4e4b3e4-4b3e-4c4e-b3e4-c4e4b3e4b3e4') // c4e4b3e4
```
